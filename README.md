# 👟 ADIDAS Landing Page

Landing Page إعلانية أنيقة لمنتج أحذية ADIDAS، مصممة باستخدام **HTML** و **CSS** فقط.  
بتتميز بتصميم عصري وخلفية ديناميكية باستخدام `clip-path`، وشكل متجاوب مع جميع أحجام الشاشات.
